¿Cómo puedo estructurar mi página principal?

(A esta estructura le llamos Layout)

✳️ ¿Que es un layout?

Layout:

Layout (en diseño web y gráfico) es la estructura o distribución visual de los elementos en una página o pantalla. Define cómo se organizan los bloques de contenido: texto, imágenes, botones, menús, etc.

🧩 En desarrollo web:

  • Header (encabezado)
  • Navbar (barra de navegación)
  • Main (contenido principal)
  • Sidebar (barra lateral)
  • Footer (pie de página)
